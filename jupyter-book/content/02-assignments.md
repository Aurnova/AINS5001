# Assignments

Course exercises follow the [`exercises/chNN/`](https://github.com/aimacode/aima/tree/main/exercises) sources from the [aima](https://github.com/aimacode/aima) book repository, rendered as pages in this site (see [Textbook chapter exercises](assignments/chapter-index.md)). Assignments are distributed via **GitHub Classroom** starter repositories.

**Schedule:** the list below is **24 class sessions** in **8 weeks of 3 meetings each** (see [Syllabus](syllabus.md) and `course/ains5001-8week-schedule.json`); each session links the AIMA slide deck (`lecture-NN`) and matching chapter exercises.

## Exercises by class week (8 weeks, 3 meetings per week, 24 class sessions)

**Format:** **8 weeks** × **3 class meetings** = **24 sessions** on the *Artificial Intelligence: A Modern Approach* (4e) slide decks. Each link targets the site file **lecture-NN** (AIMA deck / book chapter). A [full chapter index](assignments/chapter-index.md) lists every book chapter. **Exercise** numbers (C.N) follow the book; **GitHub Classroom** usually assigns a subset — follow the instructor’s rubric.
