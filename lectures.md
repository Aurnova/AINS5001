# Lessons

**Audience:** instructors using **[Aurnova AINS5001](index.html)**. The **Lessons** item in the left rail is on this same GitHub Pages site. **Dashboard** returns to the [instructor home](index.html).

The **Jupyter Book** slide index (MyST) is deployed on **this** host. Each link below uses a **path under this site** (`jupyter-book/slides/…`); there is no separate “slides” domain.

_If a link 404s locally, run a full `Deploy Pages` build, or copy `aima`’s `course/myst/_build/html` into your local `dist/jupyter-book/` to preview._
